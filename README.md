# GearAnalyzer v3.5

**GearAnalyzer** es un addon de análisis de equipo para **World of Warcraft 3.3.5a (WotLK)**, optimizado para el servidor **NaerZone**. Permite revisar en tiempo real gemas, encantamientos, glifos, talentos y BiS para todas las clases y especializaciones.

---

## 🚀 Pestañas y Funcionalidades

| Pestaña | Descripción |
|---|---|
| **Equipo** | Analiza tu gear actual, detecta piezas mejorables y muestra el estado general |
| **Gemas** | Recomendaciones de gemas por clase/spec, resumen de huecos vacíos o incorrectos |
| **Encantamientos** | Guía de encantamientos por slot con origen (reputación), check visual de estado |
| **Glifos** | Compara glifos recomendados vs equipados, con columna de estado |
| **Talentos** | Visualización del árbol de talentos para cualquier clase simulada |
| **BiS / Tops** | Lista de los mejores 6 objetos por slot. Fases: Pre-Raid, T9, T10, ICC, RS |
| **Caps** | Monitoreo de umbrales críticos: Golpe, Pericia, Haste, etc. |
| **Exportar IA** | Genera un resumen del personaje para pegar en ChatGPT, Claude o Gemini |
| **Info** | Guía integrada del addon |
| **Config** | Tamaño de fuente, iconos y opciones de minimap |

> **Selectores de Clase/Spec**: En la esquina superior derecha puedes simular cualquier clase y especialización, no solo la tuya.

---

## 🛠️ Instalación

1. Copia la carpeta `GearAnalyzer` en `World of Warcraft\Interface\Addons\`
2. Asegúrate que la carpeta se llame exactamente `GearAnalyzer`
3. Actívalo en el menú de Addons al iniciar el juego


---

## 📖 Uso Rápido

- **`Alt + Clic`** en cualquier objeto de la pestaña BiS para fijarlo como tu objetivo personal
- **Botón RESET** en BiS para volver al Top 1 por defecto
- **Shift + Clic** en un encantamiento para enlazarlo en el chat
